🐒 小猴的魔法账本 (题目描述)

小猴每天吃桃子的习惯很特别：
1. 吃掉剩下桃子的一半。
2. 没吃饱，再多吃 1 个。

到了第 n 天早上，小猴发现只剩下 1 个桃子了。
请问：第 1 天早上，一共有多少个桃子？

第3天
🍑 剩 1 个
⬅
第2天
🍑🍑🍑🍑 (1+1)×2 = 4
⬅
第1天
🍑×10 (4+1)×2 = 10

🧠 魔法原理 (Reverse Logic)

这道题的核心是 时光倒流。

🔙 正向过程：
剩下 = (原来 / 2) - 1

🔜 逆向过程（移项）：
剩下 + 1 = 原来 / 2
原来 = (剩下 + 1) × 2

我们从第 $n$ 天的 1 个桃子开始，往回推 $n-1$ 次，就能算出第 1 天的数量！

C++ 实现细节

递归函数： 我们可以定义一个函数 f(days_left)。
如果只剩 1 天（也就是第 n 天），返回 1。
否则，返回 (f(days_left - 1) + 1) * 2。

Python 实现细节

循环迭代： 相比递归，用循环倒推更直观。
从 peach = 1 开始。
循环 n-1 次，每次执行 peach = (peach + 1) * 2。

⚔️ C++ 解题代码

使用递归函数模拟倒推过程。

#include <iostream>
using namespace std;

// 递归函数：计算还剩 day 天时的桃子数
// day=1 代表最后一天，day=n 代表第一天
int get_peach(int day) {
    // 递归出口：如果是最后一天(第1天倒数)，只剩1个
    if (day == 1) {
        return 1;
    } 
    else {
        // 递推公式：(昨天的桃子 + 1) * 2
        return (get_peach(day - 1) + 1) * 2;
    }
}

int main() {
    int n;
    cin >> n;
    
    // 调用递归函数
    cout << get_peach(n) << endl;
    
    return 0;
}

🐍 Python 解题代码

无函数脚本，使用循环高效倒推。

# 1. 读入天数 n
n = int(input())

# 2. 初始化桃子数
# 第 n 天只剩下 1 个
peach = 1

# 3. 倒推 n-1 次
# 从第 n 天推回到第 1 天
for i in range(n - 1):
    # 前一天的桃子 = (今天的 + 1) * 2
    peach = (peach + 1) * 2

# 4. 输出结果
print(peach)

🌟 魔法笔记 (重点复习)

  • 🔙 逆向思维： 遇到“最后剩下...”求“最开始...”的题目，通常都可以用倒推法。
  • 🔢 数学公式： 其实这道题有一个通项公式：$3 \times 2^{n-1} - 2$。你可以试试看，结果是一样的哦！
  • 🔄 循环次数： 记得是从第 $n$ 天回到第 1 天，所以只需要推导 $n-1$ 步。
