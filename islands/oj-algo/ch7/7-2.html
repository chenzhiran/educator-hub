7.2 贪心算法的正确性证明

⚖️ 凭什么说你选的就是对的？

贪心算法就像一个极度自信的探险家，每次都不假思索地选择眼前的最优解。但直觉往往会骗人，我们怎么向别人证明这种“短视”的策略最终真的能拿到最多的宝藏呢？

在算法的世界里，我们不能光凭直觉，必须依靠严谨的数学证明。常用的两种绝招是：归纳法 和 交换论证法。

一、归纳法证明 (7.2.1)

数学归纳法就像推倒一排多米诺骨牌。我们只需证明两件事：第一块骨牌一定会倒（初始状态正确）；如果第 K 块骨牌倒了，那么第 K+1 块骨牌也一定会倒（贪心选择始终保持在最优解的路径上）。

第一步 (基础情况)： 证明当问题规模最小（比如只有一个选项）时，贪心策略显然是正确的。

第二步 (归纳假设)： 假设在进行了 K 次贪心选择后，我们目前依然处于某种“全局最优解”的包围之中。

第三步 (递推证明)： 证明我们在进行第 K+1 次贪心选择时，这个选择绝不会把我们从最优解的路径上挤出去，最终到达的终点必然还是最优解。

二、交换论证 Exchange Argument (7.2.2)

这是贪心算法最经典、最常用的证明方法。核心思想是：“不服跑个分，我把你换成我，看看结果会不会变差！”

交换论证的四步魔法：

  1. 假设存在： 假设存在一个所谓“最完美的全局最优解”（我们叫它 O），它和我们的“贪心解”（我们叫它 G）不一样。
  2. 寻找差异： 从头开始对比，找到它们之间产生分歧的第一个地方（贪心选了项目A，而最优解O在这里选了项目B）。
  3. 强制交换： 我们在那个最优解 O 里面，把 B 强行替换成贪心选择的 A。
  4. 证明不亏： 证明经过替换后，新的解不仅依然有效，而且结果不会变差（甚至变好）。由此得出结论：贪心解 G 的效果大于等于 O，所以贪心解就是最优解！

实战演练：最小化等待时间 (最短任务优先)

假设有3个人排队打水，打水需要的时间分别是 5分钟、2分钟、8分钟。贪心策略告诉我们：总是让打水时间最短的人先打！ 我们用程序实现，并看看它的图解。

C++ 最短任务优先
#include <iostream> #include <vector> #include <algorithm> using namespace std; int minWaitTime(vector<int>& tasks) { // 贪心策略：从小到大排序（最短的放前面） sort(tasks.begin(), tasks.end()); int totalWait = 0; int currentWait = 0; // 累加每个人的等待时间 for (int i = 0; i < tasks.size() - 1; i++) { currentWait += tasks[i]; totalWait += currentWait; } return totalWait; } int main() { vector<int> tasks = {5, 2, 8}; cout << "最小总等待时间: " << minWaitTime(tasks) << endl; // 输出 2 return 0; }
Python 最短任务优先
def min_wait_time(tasks): # 贪心策略：从小到大排序（最短的放前面） tasks.sort() total_wait = 0 current_wait = 0 # 累加每个人的等待时间 # 最后一个人的执行时间不计入等待时间 for i in range(len(tasks) - 1): current_wait += tasks[i] total_wait += current_wait return total_wait # 测试代码 tasks = [5, 2, 8] print(f"最小总等待时间: {min_wait_time(tasks)}") # 输出 2

三、贪心策略选择与常见误区 (7.2.3)

贪心算法最容易犯的错，就是“选错了贪心的标准”，或者将其用在了“不具备贪心选择性质”的问题上（也就是掉入了局部最优的陷阱）。

🚨 经典陷阱：0-1背包问题

假设你的背包最大容量是 50 公斤。面前有三个宝物：

  • 宝物 A：重 10kg，价值 60 💎 （性价比：6 💎/kg）
  • 宝物 B：重 20kg，价值 100 💎 （性价比：5 💎/kg）
  • 宝物 C：重 30kg，价值 120 💎 （性价比：4 💎/kg）

如果你用贪心策略：“每次挑性价比最高的！” 那么你会先拿走 A（剩余容量40），接着拿走 B（剩余容量20）。此时 C(30kg) 装不下了！你的总收益是 160 💎。

但如果你冷静下来统筹全局，放弃 A，直接选择 B 和 C。总重量 50kg 刚好装满，总收益是 220 💎！

C++ 贪心的失败尝试代码
#include <iostream> #include <vector> using namespace std; int main() { int capacity = 50; // A, B, C (已按性价比降序排列) vector<int> weights = {10, 20, 30}; vector<int> values = {60, 100, 120}; int totalValue = 0; for(int i = 0; i < 3; i++) { if(capacity >= weights[i]) { capacity -= weights[i]; totalValue += values[i]; } } cout << "贪心获得价值: " << totalValue << endl; // 输出 160，错失了最优的 220 return 0; }
Python 贪心的失败尝试代码
# A, B, C (已按性价比降序排列) weights = [10, 20, 30] values = [60, 100, 120] capacity = 50 total_value = 0 for i in range(3): if capacity >= weights[i]: capacity -= weights[i] total_value += values[i] print(f"贪心获得价值: {total_value}") # 输出 160，错失了最优的 220
