📜 任务清单 (题目描述)

系统一共会抛出 k 次宝物。每次你可以选择吃或者不吃。

游戏规则：

  • 宝物共有 n 种，每次抛出任意一种宝物的概率是均等的（都是 1/n）。
  • 吃下第 i 种宝物会获得 Pi 的分数（分数可能是负数哦！）。
  • 前置要求： 想吃第 i 种宝物，必须先把集合 Si 里的所有宝物都吃过至少一次。如果不满足条件，这件宝物就只能眼睁睁看它掉在地上消失。

计算任务： 假设你是一个超级人工智能，永远做出最明智的决定，求你在这 k 个回合中，能得到的平均分数（数学期望值）是多少？

📥 魔法输入阵：
(k次掉落，n种宝物。接着是每种宝物的分数和前置条件，以0结尾)
1 2
1 0
2 0
📤 奇迹输出阵：
(保留六位小数的平均得分)

这里有两个超级魔法需要掌握：第一是“状态压缩”，第二是“逆向期望DP”。

🎒 魔法 1：状态压缩 (用一个数字代表整个背包) 🎒

⏳ 魔法 2：逆向期望树 (从最后一轮倒推) ⏳

假设 f(i, j) 表示：在第 i 轮开始时，我们背包里的状态是 j，从这轮一直玩到游戏结束，我们能拿到的最大平均分。

现在是第 i 轮，天上随机掉下来了第 l 种宝物 (概率为 1/n)：

掉落了宝物 l
✔️ 前置条件满足！
我们可以做选择：
1. 吃掉它： 得到宝物分，且状态变为 (j加上l)
👉 分数 = P_l + f(i+1, j 加上 l)
2. 不吃它： 啥也没得，状态不变
👉 分数 = f(i+1, j)

最优策略：取这俩的 MAX 值！
❌ 前置条件不足...
我们根本吃不了它。
宝物消失，状态不变，直接进入下一轮。

👉 分数 = f(i+1, j)
💡 终极期望转移公式：

因为抛出这 n 种宝物是等概率的，所以把所有 n 种宝物的最优结果加起来，再除以 n，就是这一轮的期望得分！

f(i, j) = 累加求和 [ 面对每种宝物 l 的最佳得分 ] / n

我们从最后一轮 (第 k 轮) 往第一轮倒推计算，最后的答案就是最初始始始的期望：f(1, 0)。

📘 C++ 魔法秘籍 (知识点)

  • ⚙️ 位运算的魔法： C++ 中，j | (1 << (l - 1)) 的意思是把第 l 种宝物加入背包（把对应位置变成 1）。判断背包 j 是否满足宝物 l 的前置要求，原代码用了一个极具极客风格的位运算：!((~j) & need[l])，它等价于 (j & need[l]) == need[l]，意思是 need 中要求是 1 的位置，j 中也全都是 1。
  • 🛡️ 逆向 DP 循环： 注意看外层循环 for(i = k ; i ; i--)，它是从第 k 轮倒着算回来的。在期望 DP 中，因为未来的结果决定现在的期望，倒推是标准的正确姿势。

💻 C++ 终极法术 (代码详解)

#include <cstdio>

// sc 存分数，need 存前置要求（用二进制数压缩）
int sc[20], need[20];
// f[轮数][背包状态] = 从当前一直玩到结束的最大期望得分
double f[102][32770]; 

double max(double a, double b) {
    return a > b ? a : b;
}

int main() {
    int k, n, i, j, l, x;
    scanf("%d%d", &k, &n);
    
    // 📥 读取每种宝物的数据
    for(i = 1; i <= n; i++) {
        scanf("%d", &sc[i]);
        // 持续读取前置要求，直到遇到 0
        while(scanf("%d", &x) && x != 0) {
            need[i] |= 1 << (x - 1); // 把前置要求的位设为 1
        }
    }
    
    // 🌟 核心逆向期望 DP：从最后一轮 k 倒推回第 1 轮
    for(i = k; i > 0; i--) {
        // 遍历所有可能的背包状态 (0 到 2^n - 1)
        for(j = 0; j < (1 << n); j++) {
            // 面对随机掉落的 n 种宝物，计算每种情况
            for(l = 1; l <= n; l++) {
                // 如果背包 j 满足宝物 l 的前置要求
                if(!((~j) & need[l])) {
                    // 最优策略：在 (不吃) 和 (吃掉得分 + 新状态) 之间取最大值
                    f[i][j] += max(f[i + 1][j], f[i + 1][j | (1 << (l - 1))] + sc[l]);
                } else {
                    // 不满足条件，只能不吃
                    f[i][j] += f[i + 1][j];
                }
            }
            // 求平均值，得到在这轮的期望得分
            f[i][j] /= n;
        }
    }
    
    // 📤 输出第 1 轮开始，背包为空 (状态0) 的期望得分，保留六位小数
    printf("%.6lf\n", f[1][0]);
    return 0;
}

🐍 Python 魔法秘籍 (知识点)

  • ✨ 快速读取大法： 对于输入数据非常多的题目，使用 sys.stdin.read().split() 将全部输入一次性读入成列表，再用游标 idx 去遍历，是 Python 中避免输入超时的最佳魔法阵。
  • 🚀 状态与转移： Python 中的位运算与 C++ 完全一致。我们使用列表生成式 [[0.0] * (1 << n) for _ in range(k + 2)] 建立一个二维的 DP 表格，确保每一格都能存储浮点数。
