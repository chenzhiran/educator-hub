关于花儿实验班

六年口碑，从不打广告的编程课

花儿实验班创办于 2019 年，是扣哒世界旗下面向优秀学生的在线编程教学项目。过去六年，我们从未做过任何公开广告——每一位学员都来自家长之间的真实推荐。

我们的老师来自清华大学、北京大学等顶尖院校，他们不只是教书，更像是孩子编程路上的学长和导师。我们相信：孩子学编程最重要的事不是刷题拿奖，而是发自内心觉得"好玩"，然后在"好玩"中不知不觉地成长。

🎮

兴趣第一，绝不枯燥

依托扣哒世界全球领先的游戏化编程平台，孩子操控英雄闯关、开发自己的游戏、创造 AI 应用。2300+ 关卡持续迭代，始终保持新鲜感和挑战性。

🚀

项目产出，看得见的成果

每个阶段孩子都会做出自己的作品——能玩的游戏、可上线分享的 AI 应用、AI 短剧。不是抽象的"学了知识"，而是"我创造了什么"。

✨

因材施教，每个孩子都不一样

不做流水线教学。根据每个孩子的年龄、基础、兴趣和目标定制学习路径。喜欢做游戏、搞 AI、或挑战比赛——我们都能满足。

📚 精品小班课

系统学 Python / C++，从兴趣到竞赛一路打通

4–8 人精品小班，清北名校老师直播互动授课。孩子在游戏化闯关中系统掌握编程，培养逻辑思维，同步积累竞赛实力——全年比赛辅导包含在课程内，不额外收费。

以初中零基础孩子为例（初中生理解能力更强，进度会比小学生快）。实际教学会根据每个孩子的水平和兴趣个性化调整。

第一年 · 快速入门 + 能力进阶

Python 入门 + 游戏创作 + 进阶项目

完成计算机科学 1（地牢闯关）+ 游戏开发 1 + 计算机科学 2（森林冒险）+ 游戏开发 2 + 计算机科学 3 + 游戏开发 3。初中孩子逻辑思维更成熟，接受新概念快，加上自主练习时间更集中，一年内比小学生多推进 1–2 个地图。获得多张国际等级证书，同步完成扣哒宇宙 AI 课程。到第一年结束，已具备参加 Code Quest 和 AI 世青赛的实力。

📜 获得多张国际证书 🎮 产出多个游戏作品 🤖 AI 启蒙 + 进阶 🏆 具备参赛实力
第二年 · 高阶挑战 + 方向选择

算法深入 + 可选 C++ 竞赛方向

完成计算机科学 4 + 5 + 6，编程能力达到较高水平。根据孩子的兴趣和目标，可以选择转入 C++ 方向系统学习算法与数据结构，或者继续深入项目开发和 AI 方向。AI 课程进入高阶实战——做 AI 智能体、开发 AI 应用等。

📜 继续积累证书 🤖 AI 高阶实战 ⚡ 可选 C++ / CSP / USACO
第三年及以后 · 完全个性化

竞赛冲刺 / 独立项目 / AI 开发

根据孩子方向完全个性化——有人冲竞赛（CSP-J/S、USACO），有人做独立项目上线，有人深入 AI 开发。到这个阶段每个孩子的路径都不一样了。

💪 独立项目能力 🎯 路径完全个性化

🤖 每年都包含扣哒宇宙 AI 课程

AI 贯穿整个学习过程。从初识 AI 到动手创造 AI 应用，孩子不只是 AI 的使用者，更是 AI 的创造者——做 AI 智能体、生成 AI 短剧、开发 AI 工具。学习成果可以直接拿去参加 AI 世青赛，学和赛是打通的。

📅 上课节奏：每周精讲 + 日常自主练习

每周上课一次，其他时间孩子自行在平台闯关练习（平台有 AI 助教，卡住了一键呼出帮助）。初中生自律性更强，很多孩子课外每天会自己花 20–30 分钟闯关，进度自然就快。实际进度因人而异，有的孩子特别投入会更快，慢一些也完全没问题，老师会根据每个孩子的实际情况灵活调整节奏。

学科帮助

编程对学校学科有什么帮助？

很多家长关心：学编程到底对孩子的学校课业有没有实际帮助？以下是 7 个维度的真实反馈。

📐

数学

编程里大量用到变量、函数、坐标系、逻辑判断、循环计算——和校内数学直接相关。孩子在编程中反复使用这些概念解决实际问题，回到数学课上会觉得很熟悉、很自然。很多家长反馈孩子学了编程之后数学逻辑能力明显变强了。

🔤

英语

编程语言本身是英文的，关卡说明也有英文内容，孩子在学编程的过程中会自然积累很多专业词汇和英文阅读习惯，不需要刻意背单词。

🧠

思维能力

编程的本质是"把一个大问题拆成一步一步可执行的小步骤"。这种拆解问题的能力会迁移到所有学科——做数学应用题要分步骤，写作文要理清逻辑，物理要分析因果关系，底层都是一样的思维方式。

🤝

团队协作

Code Quest 比赛采用组队编程的形式，孩子需要和队友分工合作、沟通策略、协调配合。这种团队协作能力在学校的小组课题、未来的工作中都非常重要。日常课程中也会设置协作环节。

🎤

口语表达

Code Quest 比赛中有项目答辩环节，选手需要在 5 分钟内向评委清晰讲解团队的策略和方法。这对孩子的逻辑表达、临场应变、公众演讲能力都是很好的锻炼。

🔥

主动学习习惯

扣哒世界的游戏化设计让孩子发自内心觉得"好玩"，不需要家长催着学。因为感兴趣所以愿意主动练习，因为有即时反馈所以能坚持下去。这种"因为热爱所以坚持"的主动学习习惯一旦养成，受益的不只是编程，而是所有学科。

💪

自信心

当孩子说"这个游戏是我自己做的"，把链接发给同学玩，那种成就感和自信是其他学习方式很难给到的。自信的孩子，做什么都更有劲头。

家长关心的问题

小班课和一对一怎么选？
如果孩子希望跟同龄人一起学、系统掌握 Python/C++、享受闯关和竞赛的氛围，推荐小班课。如果孩子有明确的兴趣方向（比如想做游戏、拍 AI 短剧）、需要针对性的比赛辅导、或者时间不固定，推荐一对一。很多学员也会先上小班课打基础，再转一对一做个性化项目。
孩子不喜欢比赛，花儿实验班适合吗？
非常适合。花儿实验班的核心是"让孩子爱上编程、学到真本事、做出自己的作品"，比赛只是有兴趣的孩子的额外选项，从来不是必须的。大多数学员的日常就是在游戏闯关中学编程、在项目中创作属于自己的作品，享受创造的乐趣。
孩子零基础可以加入吗？多大适合开始？
可以。课程从零基础设计，建议 8 岁（约三年级）以上开始。入学前做能力评估，根据实际水平匹配合适的起点，不会出现"跟不上"或"太简单"的情况。
具体学什么？一年能到什么程度？
以初中零基础为例，小班课第一年完成 6 个地图（CS1 + 游戏开发 1 + CS2 + 游戏开发 2 + CS3 + 游戏开发 3），获得多张国际等级证书，掌握 Python 编程核心能力，能独立开发有一定复杂度的游戏项目，具备参加 Code Quest 和 AI 世青赛的实力。初中生理解能力更强、自学时间更集中，进度会比小学生快。一对一课则完全按孩子的目标来定，有的学员 4 个月就上线了一款完整游戏。

实际进度因人而异。有的孩子特别投入、练得多，会更快；有的孩子课外时间紧张，慢一些也完全没问题。老师会根据每个孩子的实际情况灵活调整节奏。
课后作业有人监督吗？孩子不做怎么办？
每节课结束后会布置明确的练习任务，老师在群里监督完成情况，没按时完成的孩子会单独提醒。遇到卡住的地方，群里随时可以问，老师会及时解答。我们不会"上完课就不管了"——课后跟进是教学服务的重要组成部分。
一对一课的学生作品是真实的吗？
完全真实。每个作品都由学员自己动手完成——老师负责规划路径、讲解知识、答疑指导，但代码是孩子一行一行写的，创意是孩子自己的。作品页展示的游戏可以直接试玩，AI 短剧可以直接观看，都是学员的真实产出。
